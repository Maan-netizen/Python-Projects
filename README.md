Rule-Based Chatbot 🤖
A simple chatbot built using Python that responds to user queries based on predefined rules. This project demonstrates the fundamentals of chatbot development without relying on machine learning models.

📌 Features
Responds to user inputs using if-else conditions and keyword matching.

Provides basic conversational abilities (greetings, FAQs, fallback responses).

Easy to extend with new rules and responses.

Lightweight and beginner-friendly — no external dependencies beyond Python.

🛠️ Tech Stack
Language: Python

Core Logic: Rule-based (conditional statements, keyword matching)
